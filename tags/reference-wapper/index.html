Pass a reference to a template typename argument

stackoverflow For Example: template <typename T> struct Foo { Foo(T arg) : ptr(arg) {} T ptr; }; int main() { int* a = new int(6); } There are some ways: Foo<decltype(a) &> foo(a) obscure alternative way(I do not recomand it): Foo<decltype((a))> foo(a) using std::reference_warpper Foo<std::reference_warpper<decltype(int*)>> foo(std::ref(a)); Foo<std::reference_wapper<decltype(a)>> foo(std::ref(a)); I perfer to the first way.

Store the reference in vector

std::reference_warpper is frequently used as a mechamnism to store references inside standard containers(like std::vector) std::list<int> l(10); std::vector<std::reference_wrapper<int>> v(l.begin(), l.end()); std::cout << "contents of list before:" << std::endl; for(int n : l) { std::cout << n << " "; } std::cout << std::endl; for(int& i: v) { i += 2; } std::cout << "contents of list after:" << std::endl; for(int n : l) { std::cout << n << " "; } return 0;
